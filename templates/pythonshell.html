Python Shell

Python ile yazılan programların çalıştırılması için bilgisayarınızda mutlaka bir python yorumlayıcısına ihtiyaç vardır. Yorumlayıcı sizin dosyanızı alır ve satır satır yorumlayarak byte koda dönüştürür. Oluşturulan .pyc uzantılı byte kod pc nizin belleginde çalıştırılmaya başlanır.

Python kodlarını çalıştırmak için bilgisayarınıza kurulum gerektirmeyen "Python Interaktif Shell" özellikle python programlama dilini ilk defa deneyecek olan veya kurulum yetkisi bulunmayan bilgisayarlarda python ile kodlamayı sağlamak için Google mühendisleri tarafından gerçekleştirilmiştir.

"Python Interaktif Shell" ajax teknolojisi kullanılarak Google geliştirici mühendislerinden "ryanb" tarafından kodlanmıştır.
