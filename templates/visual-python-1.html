Visual Python 1.x

Visual Tkinter 1.0 sürümü ile eklenen Python 3.x destegi artık güncel kütüphanaleri kullanmamıza imkan sağlamaktadır.

Python ile windows platformunda exe dosyaları oluşturmanın ne kadar zor olduğunu bilirsiniz. Visual Tkinter bize bu desteğide sağlıyarak py2exe modulü aracılığı ile çarçabuk exe dosyaları hazırlar. Modülün hatasız kurulduğundan ve Visual C Runtime dosyalarının sisteminizde bulunduğundan eminseniz yapmanız gerekn Ctrl+F5 tuş kombinasyonuna basmak.

Şayet py2exe ile ilgili şüpheleriniz var ise py2exe ve C Runtime ile ilgili dosyaları indirip sisteminize kurunuz.
